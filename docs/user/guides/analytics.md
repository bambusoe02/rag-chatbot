## Troubleshooting

### "No data available"

**Possible reasons:**
- New account (no queries yet)
- Date range too narrow
- Data not loaded yet

**Solutions:**
- Wait for data to accumulate
- Expand date range
- Make some queries first

### Metrics seem wrong

**Check:**
- Date range selected
- Filters applied
- Data refresh needed
- Contact support if persists
