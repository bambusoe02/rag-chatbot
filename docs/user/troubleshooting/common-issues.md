# Common Issues & Solutions

Quick fixes for frequent problems.

## Upload Issues

### "File too large"

**Problem:** File exceeds 50MB limit

**Solutions:**
1. Split file into smaller parts
2. Compress PDF (reduce quality)
3. Extract text to .txt file
4. Upgrade to Pro plan (100MB limit)

### "Unsupported file type"

**Problem:** File format not supported

**Solutions:**
1. Convert to PDF, DOCX, TXT, or MD
2. Use online converter
3. Export from original app
4. Check file extension is correct

### "Processing failed"

**Problem:** Document didn't process correctly

**Solutions:**
1. Check file isn't corrupted
2. Remove password protection
3. Ensure text is selectable (not image-only)
4. Try re-uploading
5. Contact support with file details

### Upload stuck at "Processing..."

**Problem:** Processing takes > 5 minutes

**Solutions:**
1. Wait 2-3 more minutes (large files)
2. Refresh page
3. Check file size
4. Try uploading again
5. Contact support if persists

## Query Issues

### "No documents found"

**Problem:** No relevant documents for query

**Solutions:**
1. Upload documents first
2. Check documents processed successfully
3. Try broader question
4. Use different search mode
5. Verify document contains answer

### Answer seems wrong

**Problem:** Answer doesn't match document

**Solutions:**
1. Check cited sources
2. Try different search mode
3. Rephrase question more specifically
4. Verify document quality
5. Ask follow-up for clarification

### Answer too general

**Problem:** Answer lacks detail

**Solutions:**
1. Be more specific in question
2. Ask for examples
3. Specify section/chapter
4. Break into smaller questions
5. Use keyword search for exact terms

### Slow response times

**Problem:** Queries take > 10 seconds

**Solutions:**
1. Check internet connection
2. Try smaller documents
3. Reduce top_k results
4. Use keyword search (faster)
5. Check server status

## Account Issues

### Can't login

**Problem:** Login fails

**Solutions:**
1. Check username/password spelling
2. Reset password
3. Try incognito/private mode
4. Clear browser cookies
5. Contact support

### "Account locked"

**Problem:** Too many failed login attempts

**Solutions:**
1. Wait 15 minutes
2. Reset password
3. Contact support to unlock
4. Check email for unlock link

### Forgot password

**Problem:** Can't remember password

**Solutions:**
1. Click "Forgot Password"
2. Enter email
3. Check email for reset link
4. Click link and set new password
5. Check spam folder if no email

## Performance Issues

### App is slow

**Problem:** Interface responds slowly

**Solutions:**
1. Refresh page
2. Clear browser cache
3. Close other tabs
4. Try different browser
5. Check internet speed

### Queries timeout

**Problem:** Request times out

**Solutions:**
1. Check server status
2. Try simpler question
3. Reduce document size
4. Wait and retry
5. Contact support

### High memory usage

**Problem:** Browser uses too much memory

**Solutions:**
1. Close other tabs
2. Clear chat history
3. Refresh page
4. Use lighter browser
5. Restart browser

## Document Issues

### Can't find document

**Problem:** Document not in list

**Solutions:**
1. Check correct user account
2. Use search box
3. Check upload completed
4. Look in analytics
5. Verify not deleted

### Document shows as "Processing"

**Problem:** Stuck in processing state

**Solutions:**
1. Wait 5 minutes
2. Refresh page
3. Check file size
4. Re-upload if needed
5. Contact support

### Wrong document content

**Problem:** Answer references wrong document

**Solutions:**
1. Check document names
2. Verify correct file uploaded
3. Delete and re-upload
4. Check for duplicate names
5. Use document preview

## API Issues

### "Invalid API key"

**Problem:** API key not working

**Solutions:**
1. Check key copied correctly
2. Verify key not revoked
3. Check header name (`X-API-Key`)
4. Verify key belongs to account
5. Create new key if needed

### "Rate limit exceeded"

**Problem:** Too many API requests

**Solutions:**
1. Wait for reset period
2. Check rate limit in dashboard
3. Implement request queuing
4. Upgrade plan for higher limit
5. Use caching

### API response errors

**Problem:** API returns errors

**Solutions:**
1. Check request format
2. Verify authentication
3. Check endpoint URL
4. Review error message
5. Check API documentation

## Browser Issues

### Page won't load

**Problem:** Blank page or error

**Solutions:**
1. Refresh page (Ctrl+R / Cmd+R)
2. Clear browser cache
3. Try incognito mode
4. Try different browser
5. Check JavaScript enabled

### Features not working

**Problem:** Buttons/links don't work

**Solutions:**
1. Refresh page
2. Clear cache
3. Disable browser extensions
4. Update browser
5. Try different browser

### Styling looks wrong

**Problem:** Page layout broken

**Solutions:**
1. Refresh page
2. Clear cache
3. Disable ad blockers
4. Check browser zoom (100%)
5. Update browser
