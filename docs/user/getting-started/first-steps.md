# First Steps with RAG Chatbot

Welcome! Let's get you started in just 5 minutes.

## Step 1: Create Account (2 minutes)

### Web App

1. Go to https://rag-chatbot.com
2. Click **"Register"**
3. Fill in:
   - Username (letters, numbers, underscore)
   - Email address
   - Password (min 8 chars, uppercase, lowercase, number)
4. Click **"Create Account"**
5. Check email for verification (if enabled)

### Self-Hosted

1. Go to http://localhost:8501
2. Same registration process
3. First user becomes admin automatically

## Step 3: Ask Your First Question (1 minute)

### In the Chat Box

1. Type your question in the input box
2. Examples:
   - "What is the main topic of this document?"
   - "Summarize the key points"
   - "What does the author say about X?"
3. Press **Enter** or click **"Send"**
4. Wait 2-5 seconds for answer
5. See sources cited below answer

### Search Modes (Optional)

Choose search mode before asking:
- **Hybrid** (Recommended) - Best of both worlds
- **Semantic** - Understands meaning
- **Keyword** - Exact word matching
